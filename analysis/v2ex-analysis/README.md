# V2EX Analysis Dashboard

Vue 3, TypeScript, Vite, and Plotly dashboard for JSON generated by `../build_analytics.py`.

```bash
cd analysis/v2ex-analysis
npm install
npm run dev -- --host 0.0.0.0
npm run build
```

Generated `dynamic-*.json` files are served from `public/`. Do not edit them manually; rebuild them from the root `v2ex.sqlite` with:

```bash
.venv/bin/python analysis/build_analytics.py
```
